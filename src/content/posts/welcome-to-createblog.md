CreateBlog is a starter repository for people who want a personal blog without giving up control.

You keep the content local, write in Markdown, push to GitHub, and let GitHub Pages publish the site.

## What this starter gives you

- A homepage with a configurable hero and featured post area
- Tag index pages generated from your Markdown frontmatter
- Multiple visual templates that can be swapped from a single config file
- Click effects and accent colors for lightweight personalization

## How you customize it

Edit `blog.config.json` to change your site title, navigation, GitHub username, template, accent color, and click effect.

Then create new posts in `src/content/posts/` and push them to your own repository.

## Why Markdown first

Markdown keeps authorship simple. You can draft locally, review changes in Git, and publish without a database or admin panel.
